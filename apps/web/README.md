# Web app README

This folder contains a minimal Next.js website for the Royal Chess project.

Run locally:
1. cd apps/web
2. pnpm install
3. pnpm dev (or npm run dev if using npm)

Build for production:
1. pnpm build
2. pnpm start

Deploy:
- Deploy to Vercel by connecting this repository and selecting the apps/web directory as the project root.
- Or deploy to any static host that supports Next.js.
